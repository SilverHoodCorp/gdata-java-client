Instructions for the Google Prediction API Sample

Browse Online

  • Browse Source, or main file PredictionSample.java

Installation Instructions

Prerequisite: you must first install the gdata library project.

Prerequisite: you must first install Java 6.

Important: after checking out the project, and before compiling and running it, you need to edit src/com/google/api/client/sample/prediction/ClientLoginCredentials.java to enter a username and password that has access to the Google Prediction API. Otherwise, you will get a 403 Forbidden exception. You will also need to upload the training data to Google Storage and edit the OBJECT_PATH.

If you need commit access to this sample, you first need to be added as a Committer.

Command-Line

You will need to first install Subversion and Maven.

  • SVN checkout
    • Read only: svn checkout http://gdata-java-client.googlecode.com/svn/branches/2/sample/prediction/prediction-sample <some path>/prediction-sample
    • Commit access: svn checkout https://gdata-java-client.googlecode.com/svn/branches/2/sample/prediction/prediction-sample <some path>/prediction-sample
  • cd <some path>/prediction-sample
  • Compile: mvn compile
  • Run: mvn -q exec:java

Eclipse 3.5/3.6

You will need to first install Eclipse, the Subversion plugin, and the Maven plugin.

  • SVN checkout
    • File > Import...
    • Select "SVN > Project from SVN" and click "Next"
    • "Checkout Project from SVN repository" or "Enter Repository Location Information"
      • URL:
        • Read only: http://gdata-java-client.googlecode.com/svn
        • Commit access: https://gdata-java-client.googlecode.com/svn
      • Click on "Next"
    • Select Resource
      • URL:
        • Read only: http://gdata-java-client.googlecode.com/svn/branches/2/sample/prediction/prediction-sample
        • Commit access: https://gdata-java-client.googlecode.com/svn/branches/2/sample/prediction/prediction-sample
      • Click on "Head revision" and click on "Finish"
    • Check Out As
      • Click on "Finish"
  • Run
    • Right-click on project prediction-sample
    • Run As > Java Application
