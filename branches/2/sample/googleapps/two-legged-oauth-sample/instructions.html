    Command-Line Instructions

    You will need to first install Subversion.

    Google engineers only: if you need commit access, you first need to be added as a Committer.

    • SVN checkout
      • Read only: svn checkout http://gdata-java-client.googlecode.com/svn/tags/2.2.1-alpha/sample/googleapps/two-legged-oauth-sample <some path>/two-legged-oauth-sample
      • Commit access: svn checkout https://gdata-java-client.googlecode.com/svn/branches/2/sample/googleapps/two-legged-oauth-sample <some path>/two-legged-oauth-sample
    • cd <some path>/two-legged-oauth-sample
    • Compile: ant
    • Run: ant run -Dargs" -comsumer_key=bla.com ...etc"
    • For details on command line args, see comments in main file TwoLeggedOauthSample.java
