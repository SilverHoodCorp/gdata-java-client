Setup Instructions for the Google Language API for Diacritization Sample

Browse Online

  • Browse Source, or main file DiacritizationSample.java

Installation Instructions

Prerequisite: you must first install the gdata library project.

Prerequisite: you must first install Java 6.

If you need commit access, you first need to be added as a Committer.

Command-Line

You will need to first install Subversion and Maven.

  • SVN checkout
    • Read only: svn checkout http://gdata-java-client.googlecode.com/svn/branches/2/sample/diacritization/diacritization-sample <some path>/diacritization-sample
    • Commit access: svn checkout https://gdata-java-client.googlecode.com/svn/branches/2/sample/diacritization/diacritization-sample <some path>/diacritization-sample
  • cd <some path>/diacritization-sample
  • Compile: mvn compile
  • Run: mvn -q exec:java

Eclipse 3.5/3.6

You will need to first install Eclipse, the Subversion plugin, and the Maven plugin.

  • SVN checkout
    • File > Import...
    • Select "SVN > Project from SVN" and click "Next"
    • "Checkout Project from SVN repository" or "Enter Repository Location Information"
      • URL:
        • Read only: http://gdata-java-client.googlecode.com/svn
        • Commit access: https://gdata-java-client.googlecode.com/svn
      • Click on "Next"
    • Select Resource
      • URL:
        • Read only: http://gdata-java-client.googlecode.com/svn/branches/2/sample/diacritization/diacritization-sample
        • Commit access: https://gdata-java-client.googlecode.com/svn/branches/2/sample/diacritization/diacritization-sample
      • Click on "Head revision" and click on "Finish"
    • Check Out As
      • Click on "Finish"
  • Run
    • Right-click on project diacritization-sample
    • Run As > Java Application
