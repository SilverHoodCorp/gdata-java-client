Setup Instructions for the Google Discovery API Sample

Browse Online

  • Browse Source, or main file DiscoverySample.java

Installation Instructions

Prerequisite: you must first install the gdata library project.

If you need commit access, you first need to be added as a Committer.

Eclipse 3.5/3.6

You will need to first install Eclipse, the Subversion plugin, and the Maven plugin.

  • SVN checkout
    • File > Import...
    • Select "SVN > Project from SVN" and click "Next"
    • "Checkout Project from SVN repository" or "Enter Repository Location Information"
      • URL:
        • Read only: http://gdata-java-client.googlecode.com/svn
        • Commit access: https://gdata-java-client.googlecode.com/svn
      • Click on "Next"
    • Select Resource
      • URL:
        • Read only: http://gdata-java-client.googlecode.com/svn/branches/2/sample/discovery/discovery-sample
        • Commit access: https://gdata-java-client.googlecode.com/svn/branches/2/sample/discovery/discovery-sample
      • Click on "Head revision" and click on "Finish"
    • Check Out As
      • Click on "Finish"
  • Run
    • Right-click on project discovery-sample
    • Run As > Java Application
    • Select "DiscoverySample" and click "OK"
    • Run > Run Configurations...
    • select "Java Application" > "Discovery Sample"
    • In Arguments tab, Program arguments: discover buzz
    • click "Run"

Command-Line

You will need to first install Subversion and Maven.

  • SVN checkout
    • Read only: svn checkout http://gdata-java-client.googlecode.com/svn/branches/2/sample/discovery/discovery-sample <some path>/discovery-sample
    • Commit access: svn checkout https://gdata-java-client.googlecode.com/svn/branches/2/sample/discovery/discovery-sample <some path>/discovery-sample
  • cd <some path>/discovery-sample
  • Assemble: mvn assembly:assembly
  • Run: google
