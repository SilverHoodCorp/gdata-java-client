GData Android Client Library version 2.2.0-alpha

See: http://code.google.com/p/gdata-java-client/wiki/Version2

This library in general supports all Java 2 Standard Edition environments, including Android and App Engine. These directions cover the Android environment. There is a separate larger download for all other Java environments available with a readme file customized for those environments.

Android

When building an application for Android, you will want to minimize the size of your application by only including the jars you need from this library. Dependent jars are found in the dependencies folder. Service-specific Google API jars are found in the googleapis folder.

  • gdata-core-android-2.2.0-alpha.jar: required
  • gdata-xml-android-2.2.0-alpha.jar: needed for user-defined XML data models
  • gdata-json-android-2.2.0-alpha.jar and dependencies/jackson-core-asl-1.5.0.jar: needed for user-defined JSON data models
  • googleapis-<version>-android-2.2.0-alpha.jar: optional service-specific Google API jars that provides minimal information like the authTokenType to use with AccountManager and the GData version to use with GoogleTransport.setGDataVersion().
  • googleapis-<version>-atom-android-2.2.0-alpha.jar: optional service-specific Google API jars that provides the XML namespace information to use for example with AtomHttpParser.namespaceDictionary.
