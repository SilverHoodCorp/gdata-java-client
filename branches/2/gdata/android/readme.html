Google Data Android Client Library version 2.2.1-alpha

See: http://code.google.com/p/gdata-java-client/wiki/Version2

The Google Data (GData) client library version 2 in general supports all Java 2 Standard Edition environments, including Android and App Engine. This download is only for the Android environment.

The download for all other Java environments can be found at: gdata-2.2.1-alpha.zip

Android

While developing an Android application, it is convenient to use gdata-2.2.1-alpha.jar. Unfortunately, its large size makes it unwieldy, especially since you are likely to only use a relatively small subset of the library. If multiple Android applications use this library on an Android device with limited application space, the problem grows significantly. Thus, to minimize the library size, use the packaged jars When building an application for Android. You can figure out the package dependencies by looking at end of the package description in the JavaDoc. Note that package dependencies may change in future versions, so beware of such changes when upgrading to a newer version.

Note: jackson-core-asl-1.5.2.jar is a third-party library needed when working with JSON. It can be downloaded from here.
