Release Process

  • Compile distribution jar 
    cd gdata-java-client/gdata
ant all
    (make sure all tests pass)
  • Upload to http://code.google.com/p/gdata-java-client/downloads/entry
    • Summary: Experimental GData Java Client Library Version 2.2.0 Alpha (not backwards compatible with version 1.X)
    • Labels: Type-Archive, OpSys-All, and Featured
    • File: gdata-java-client/gdata/build-bin/gdata-java-2.2.0-alpha.zip
    • click Submit file
  • Create SVN Tag 
    svn cp -m "Tagging 2.2.0-alpha" https://gdata-java-client.googlecode.com/svn/branches/2 https://gdata-java-client.googlecode.com/svn/tags/2.2.0-alpha
  • Upload JavaDoc into Tag 
    svn checkout https://gdata-java-client.googlecode.com/svn/tags/2.2.0-alpha /tmp/tag
cd gdata-java-client/gdata
cp -R build-bin/dist/javadoc /tmp/tag/javadoc
cd /tmp/tag
svn add javadoc
cd javadoc
find . -name "*.html" -type f -print0 | xargs -0 svn propset svn:mime-type text/html
find . -name "*.css" -type f -print0 | xargs -0 svn propset svn:mime-type text/css
cd ..
svn commit -m "JavaDoc for version 2.2.0-alpha"
cd ..
rm -rf tag
  • Update http://code.google.com/p/gdata-java-client/w/edit/Version2 if necessary
