Eclipse 3.5 Instructions

You will need to first install Eclipse 3.5, the Subversion plugin, and the Android plugin.

Google engineers only: if you need commit access, you first need to be added as a Committer.

  • Preferences: Android; setup SDK location
  • File > Import...
  • SVN > Project from SVN
  • Next
  • URL:
    • Read only: http://gdata-java-client.googlecode.com/svn
    • Commit access: https://gdata-java-client.googlecode.com/svn
  • Next
  • URL:
    • Read only: http://gdata-java-client.googlecode.com/svn/tags/2.3.0-alpha/gdata
    • Commit access: https://gdata-java-client.googlecode.com/svn/branches/2/gdata
  • Click on Head revision
  • Finish
  • Finish

Command-Line Instructions

You will need to first install Subversion.

Google engineers only: if you need commit access, you first need to be added as a Committer.

  • SVN checkout
    • Read only: svn checkout http://gdata-java-client.googlecode.com/svn/tags/2.3.0-alpha/gdata <some path>/gdata
    • Commit access: svn checkout https://gdata-java-client.googlecode.com/svn/branches/2/gdata <some path>/gdata
  • cd <some path>/gdata
  • Incremental compile: ant
  • Clean, compile, distribute, and test: ant all
