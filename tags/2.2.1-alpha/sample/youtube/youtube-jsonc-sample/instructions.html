Setup Instructions for the YouTube Data API Sample (JSON-C)

Browse Online

  • Browse Source, or main file YouTubeJsoncSample.java

Eclipse 3.5 Instructions

You will need to first install Eclipse 3.5, and the Subversion plugin.

Google engineers only: if you need commit access, you first need to be added as a Committer.

  • File > Import...
  • SVN > Project from SVN
  • Next
  • URL:
    • Read only: http://gdata-java-client.googlecode.com/svn
    • Commit access: https://gdata-java-client.googlecode.com/svn
  • Next
  • URL:
    • Read only: http://gdata-java-client.googlecode.com/svn/tags/2.2.1-alpha/sample/youtube/youtube-jsonc-sample
    • Commit access: https://gdata-java-client.googlecode.com/svn/branches/2/sample/youtube/youtube-jsonc-sample
  • Click on Head revision
  • Finish
  • Finish

Command-Line Instructions

You will need to first install Subversion.

Google engineers only: if you need commit access, you first need to be added as a Committer.

  • SVN checkout
    • Read only: svn checkout http://gdata-java-client.googlecode.com/svn/tags/2.2.1-alpha/sample/youtube/youtube-jsonc-sample <some path>/youtube-jsonc-sample
    • Commit access: svn checkout https://gdata-java-client.googlecode.com/svn/branches/2/sample/youtube/youtube-jsonc-sample <some path>/youtube-jsonc-sample
  • cd <some path>/youtube-jsonc-sample
  • Compile: ant
  • Run: ant run
